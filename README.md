# Akshara Singh | Brand Communications & Social Media Portfolio

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_Website-success?style=for-the-badge&logoColor=white)](https://aksharasingh-portfolio.surge.sh)

A high-performance, interactive personal portfolio website designed to showcase my expertise in brand communication, audience-first social strategy, and execution-driven marketing. Built specially for the TestGorilla Brand Communications & Social Media Specialist application.

## 🚀 Features

- **High-End Scroll Animations:** Uses Framer Motion for premium, optimized scroll-linked interactions and text reveals.
- **Hardware-Accelerated UX:** Custom GPU compositing hints ensuring 0% repainting and perfectly smooth transitions.
- **Comprehensive Case Studies:** Detailed breakdown of D2C and B2B workflows (Throttle Theory, Cashfree, PhonePe).
- **Social Media Strategy & Content Samples:** Specific mockups for LinkedIn, Instagram, Newsletter, and Community Engagement for TestGorilla.
- **First 30 Days Plan:** A structured onboarding approach formatted as an interactive timeline.

## 💻 Technologies Used

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Deployment:** Surge (Static Export)
